🚀 Machete Appwise: POO & Canvas API

1. Programación Orientada a Objetos (POO)

El Molde (Clase y Constructor)

La fábrica para crear múltiples objetos con new.

class Entidad {
  constructor(x, y) {
    this.x = x; // 'this' es ESTA instancia
    this.y = y;
  }
  moverse() {
    this.x += 5;
  }
}
const jugador = new Entidad(10, 20);

Privacidad (Getters y Setters)

Proteger variables con # y controlarlas.

class Banco {
  #saldo = 1000; // Variable privada

  get verSaldo() {
    return this.#saldo;
  }
  set modificarSaldo(monto) {
    if (monto > 0) this.#saldo = monto;
  }
}
// Se usan SIN paréntesis:
banco.modificarSaldo = 5000;

Herencia (extends & super)

Crear una clase hija basada en una clase padre.

class Enemigo extends Entidad {
  constructor(x, y, dano) {
    super(x, y); // Llama al constructor padre
    this.dano = dano;
  }
  atacar() {
    console.log("¡Daño: " + this.dano + "!");
  }
}

2. Canvas 2D API & Motores de Juego

1. Configuración Inicial

Atrapar el lienzo y pedir el contexto.

/** @type {HTMLCanvasElement} */
const canvas = document.querySelector("#lienzo");
const ctx = canvas.getContext("2d");

2. El Game Loop (Vital)

La ilusión de movimiento: borrar, actualizar y redibujar.

function loop() {
  // 1. Borrar todo
  ctx.clearRect(0, 0, canvas.width, canvas.height);

  // 2. Actualizar variables
  // 3. Dibujar

  // 4. Repetir (sincronizado con el monitor)
  requestAnimationFrame(loop);
}
loop();

3. Dibujar Rectángulos

Personajes retro y plataformas.

// Relleno
ctx.fillStyle = "blue";
ctx.fillRect(x, y, ancho, alto);

// Contorno
ctx.strokeStyle = "red";
ctx.lineWidth = 3;
ctx.strokeRect(x, y, ancho, alto);

4. Dibujar Círculos

Pelotas y ruedas.

ctx.beginPath();
// arc(x, y, radio, anguloInicio, anguloFin)
ctx.arc(x, y, 10, 0, Math.PI * 2);
ctx.fillStyle = "white";
ctx.fill();
ctx.closePath();

5. Dibujar Textos

Para marcadores y Game Over.

ctx.fillStyle = "#00ffcc";

// fillText("Texto", X, Y)
